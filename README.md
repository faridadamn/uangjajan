# UangJajan

Aplikasi HTML statis untuk catatan keuangan anak.

## Deploy GitHub Pages

Folder ini siap dipush ke repository GitHub Pages. File utama: `index.html`.

### Opsi A — User site
Buat repo bernama:

```text
<username>.github.io
```

Push isi folder ini ke branch `main`. URL akan menjadi:

```text
https://<username>.github.io/
```

### Opsi B — Project site
Buat repo misalnya:

```text
uangjajan
```

Push isi folder ini ke branch `main`, lalu aktifkan **Settings → Pages → Deploy from a branch → main / root**. URL akan menjadi:

```text
https://<username>.github.io/uangjajan/
```
